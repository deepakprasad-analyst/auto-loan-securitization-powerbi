# Auto Loan Securitization Analytics Dashboard
### Power BI | DAX | Power Query | Financial Analytics

## Project Overview
Developed a comprehensive 4-page Auto Loan Securitization Analytics 
Dashboard for Zetheta Algorithms Pvt. Ltd. to analyze a loan portfolio 
of 500 loans worth ₹546M original pool balance.

## Dashboard Pages

### 1. Executive Summary & Pool Overview
- Total Loans: 500 | Original Pool Balance: ₹546M
- Current Pool Balance: ₹318M | Pool Factor: 58.19%
- Average Loan Size: ₹1.1M | WAC: 10.95%
- Top 10 State Exposure & Region-wise analysis

### 2. Pool Stratification & Risk Analysis
- LTV Distribution across buckets (61–90%)
- Employment Mix: Professional, Salaried, Self-Employed
- CIBIL Score Distribution (650 to 800+)
- New vs Used Vehicle Mix (88.55% new vehicles)

### 3. Delinquency Analysis
- 30+ DPD Rate: 15.60% | 60+ DPD: 11.40% | 90+ DPD: 6.60%
- Total Current Balance: $4.83bn
- DPD Bucket Distribution & Monthly DPD Trend
- Transition Analysis: Cure 20.59%, Roll 28.18%, Static 51.23%

### 4. Credit Risk & IFRS-9 Analysis
- Total EAD: ₹326.16M | ECL Provision: ₹11.66M
- Average LGD: 50.34% | Default Rate: 1.80% | Avg PD: 8.18%
- IFRS9 Stage Distribution & PD vs LGD analysis

## Key Features
- Interactive slicers (Region, Vehicle Type, Employment, State)
- Drill-through pages for deep-dive analysis
- DAX measures for DPD rates, ECL, PD, LGD calculations
- Dynamic KPI cards with date range filters
